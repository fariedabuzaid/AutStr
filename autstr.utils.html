autstr.utils.automata_tools.expand(dfa, n, pos)¶

Takes an automaton that recognizes a k-ary presentation R and expands it to a n-ary presentation S, n>=k, with S = {(x_1,…, x_n) | (x_{pos[0]},…, x_{pos[k-1]}) in R}

Parameters:

  • dfa (DFA) – The automaton

  • n (int) – The arity of the result

  • pos (List) – where to encode the original relation

Return type:

DFA

Returns:

Automaton recognizing {(x_1,…, x_n) | (x_{pos[0]},…, x_{pos[k-1]}) in L(dfa)}

Generator over the language that is represented by a DFA. Provides functionality for decoding of words by user defined decoder functions. Iterates through the words in length-lexicographic order.

autstr.utils.automata_tools.projection(dfa, i)¶

Takes an automaton that recognizes a k-ary relation R and a position i<k and returns and automaton that recognizes the relation R_{-i} = {(x_1,…, x_{i-1}, x_{i+1},…, x_k) | exists x_i: (x_1,…, x_n) in R}

Parameters:

  • dfa (DFA) – The automaton

  • i (int) – the position

Return type:

Return type:
